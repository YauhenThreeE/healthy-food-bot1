# healthy-food-bot

Telegram bot for healthy meal ordering, nutrition tracking, and lifestyle support.

## MVP функции

- онбординг (цели, ограничения)
- каталог еды
- заказ наборов
- инструкции приготовления
- дневник питания
- напоминания

## команды

/start – старт
/menu – меню
/journal – дневник
/orders – заказы

## запуск

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python bot.py
```
